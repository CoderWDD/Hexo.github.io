什么是 Retrofit

A type-safe HTTP client for Android and Java

与 OkHttp 的关系

源码入口

从 retrofit 的使用步骤中，可以知道，retrofit.create(service) 是整个 retrofit 的入口

create函数

  • create 方法中，通过 动态代理 的方式，对传入的 service 进行代理，并进行一定检查
  • 可以看出，create() 会先调用 validateServiceInterface(service) 进行判断，如果不符合要求，就会报错，这也是保证 Retrofit type safe 的一个手段
  • 通过了类型判断后，就会对其进行动态代理了，从代码可以看出，只代理了我们自定义的方法，而 Object 类的方法是不会被代理的
  • 最后，根据方法是否是 default 方法，进行了不同的调用，需要注意的是，这里的 loadServiceMethod(method) 里面有个 缓存机制，也是用于获取实际要运行的方法的

validateServiceInterface 函数

  • 从代码即注释可以看出，validateServiceInterface() 的作用就跟函数名字一样，主要是用于校验所代理是 service 是否合理

loadServiceMethod 函数
