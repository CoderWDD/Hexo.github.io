MVC 简介

MVC 三个字母分别代表什么

  • M：Model，包含了一些处理逻辑和数据定义
  • V：View，用于处理用户输入和显示，负责用户交互
  • C：Controller，连接 Model 和 View 层，接收来自 View 的事件，并分发给不同的 Model，最后调用 View 层的方法，更新 View

MVC 三层之间的联系

  • Model 层负责定义数据类型，并进行相关的逻辑处理，并在处理完数据后，把结果通知 View 层
  • View 层负责与用户交互，并把相关的事件传递给 Controller 层
  • Controller 层负责接收来自 View 层的事件，并调用 Model 层的处理，把结果组合后返回给 View 层使用

总结
