简介

A Thread that has a Looper. The Looper can then be used to create Handlers.
一个自带 Looper 的 Thread

作用

HandlerThread 继承自 Thread，且内部自带一个 Looper，所以，可以用于处理耗时任务，特别是如果要在同一个线程中执行多次耗时任务时，用 HandlerThread 是个不错的选择
